# Grammar Master

A Flutter app for learning English grammar through structured theory and AI-generated quizzes. Every quiz is freshly created by an LLM — no two sessions are the same.

---

## Features

- **15 grammar topics** — Present Simple through Relative Clauses, each with structured theory, key rules, and examples
- **AI-powered quizzes** — 20 multiple-choice questions generated on demand by Llama 3.3 via Groq
- **Instant explanations** — wrong answers trigger a bottom sheet with the correct answer and a grammar rule explanation
- **Local progress tracking** — scores and completion status saved with Isar (no cloud, no account required)
- **Result review** — after each quiz, review every question you got wrong with expandable explanations
- **Offline-first** — only the quiz generation requires internet; all other data lives on device

## Getting Started

### Prerequisites

- Flutter SDK >= 3.0
- Android Studio or VS Code with Flutter extension
- A free [Groq API key](https://console.groq.com)

### 1. Clone and install dependencies

```bash
git clone https://github.com/YOUR_USERNAME/grammar_learning__app.git
cd grammar_learning__app
flutter pub get
```

### 2. Generate Isar database code

```bash
dart run build_runner build --delete-conflicting-outputs
```

### 3. Add your API key

Copy the example config file and add your Groq key:

```bash
cp lib/core/constants/api_constants.example.dart lib/core/constants/api_constants.dart
```

Then open `lib/core/constants/api_constants.dart` and replace `YOUR_GROQ_API_KEY_HERE` with your key from [console.groq.com](https://console.groq.com).


### 4. Run the app

```bash
flutter run
```

## How to Use

1. **Home screen** — browse the 15 topic cards. Completed topics show your best score percentage.
2. **Theory screen** — tap any topic to read the grammar structure, rules, and examples before testing yourself.
3. **Start Quiz** — tap the button at the bottom of the theory screen. The AI generates 20 fresh questions in a few seconds.
4. **Answer questions** — tap an option. Correct answers turn green; wrong answers turn red and a bottom sheet pops up with an explanation.
5. **Results** — see your score out of 20, your accuracy, and a review of every question you got wrong (tap to expand).
6. **Progress** — your best score for each topic is saved locally and shown on the home screen.

## Grammar Topics

| # | Topic | Focus |
|---|---|---|
| 1 | Present Simple | Habits, facts, schedules |
| 2 | Present Continuous | Ongoing actions, temporary situations |
| 3 | Past Simple | Completed past actions |
| 4 | Past Continuous | Background/interrupted past actions |
| 5 | Present Perfect | Past with present relevance |
| 6 | Past Perfect | Earlier past before another past event |
| 7 | Future: will & going to | Plans vs. spontaneous decisions |
| 8 | Zero & First Conditional | Real and possible conditions |
| 9 | Second & Third Conditional | Hypothetical and unreal situations |
| 10 | Modal Verbs | Ability, obligation, advice, possibility |
| 11 | Passive Voice | Focus on action, not the doer |
| 12 | Articles: a, an, the | Definite vs. indefinite reference |
| 13 | Prepositions of Time & Place | at / on / in |
| 14 | Reported Speech | Tense backshift, pronoun changes |
| 15 | Relative Clauses | Defining vs. non-defining |

## Design Decisions

- **Quiz questions are never stored** — they exist only in memory during a session and are discarded on completion, keeping the database small and fast.
- **Isar over Firebase** — fully local, zero latency reads, no account or internet required for progress tracking.
- **Riverpod `autoDispose`** — the quiz state provider is scoped to the quiz screen and auto-disposed when you leave, preventing stale data.
- **JSON mode** — Groq's `response_format: {type: "json_object"}` forces the LLM to return valid, parseable JSON every time.
